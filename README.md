# CRUD-Estilo-Trello
Sistema CRUD de tasks desenvolvido com Node.js/Express no backend e React/Vite no frontend. Inclui operações completas de criação, leitura, atualização e exclusão de tasks, com interface moderna e responsiva.

## Pastas do projeto
- **Backend:** `CRUD - Gerenciador de tasks back` — `npm install`, `npx prisma generate`, `npm run dev`
- **Frontend:** `CRUD - Gerenciador de tasks front` — `npm install`, `npm run dev`
